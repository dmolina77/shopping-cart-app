### 🛒 MercadoDemo - Carrito de Compras con JavaScript y Bootstrap 5

### Descripción

Proyecto de práctica para reforzar conceptos de JavaScript, incluyendo:

- Manipulación del DOM.

- Eventos.

- Arrays y objetos.

- Consumo de APIs.

- LocalStorage.

La aplicación muestra productos desde una API pública (DummyJSON) y permite agregarlos o eliminarlos del carrito de compras.

### Objetivo

El propósito principal del proyecto es practicar lógica de programación y desarrollo Frontend sin frameworks.

### Características

- Listado dinámico de productos.

- Carrito interactivo.

- Cálculo automático del total.

- Persistencia de datos con LocalStorage.

- Diseño responsive con Bootstrap 5.

### Tecnologías

- HTML5, JavaScript ES6+, Bootstrap 5 y DummyJSON API
